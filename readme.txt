== Description ==

With this widget you can add an option to see the top posts on your blog appear on the sidebar.
Features:

*   configurable number of displayed records;
*   conclusion quote, or just the title;
*   display in a list or any other form (the tags are set);
*   you can set the display of posts from only certain categories;
*   you can specify whether or not the number of comments.

== Installation ==

1. Unzip files.
2. Upload 'top-posts-widget' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Activate the widget through the 'Widgets' menu in WordPress

== Frequently Asked Questions ==

Yes with this plugin you can easily change parameters to suit your liking and the look of your blog.
